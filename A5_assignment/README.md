# A5 Assignment Site

## Overview
This repository contains the **A5 Assignment** web page, built as part of the course portfolio. The site showcases a modern, dark‑themed UI with a responsive layout, custom typography, and interactive elements.

## Features
- **Responsive Design**: Adapts gracefully to mobile, tablet, and desktop viewports using CSS Grid and Flexbox.
- **Premium Aesthetics**: Dark mode with vibrant accent colors, smooth gradients, and subtle micro‑animations for a premium feel.
- **Custom Typography**: Uses Google Fonts (Inter) for clean, modern text rendering.
- **Interactive Components**: Includes navigation toggles, hover effects, and animated cards.

## Getting Started
1. **Clone the repository** (or copy the folder):
   ```
   git clone <repo-url>
   ```
2. **Open the site locally**:
   - Navigate to the `A5_assignment` folder.
   - Open `index.html` in a web browser (e.g., Chrome, Edge).

   No server is required; the page runs as a static HTML file.

## Project Structure
```
A5_assignment/
├─ index.html        # Main entry point of the site
├─ styles.css        # Core styling (grid, typography, colors)
├─ script.js         # JavaScript for interactive behavior
└─ assets/           # Images, icons, and other media assets
```
